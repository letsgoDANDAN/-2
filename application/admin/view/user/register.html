图书馆阅览须知

一、图书馆阅览室为开架阅览，对全体师生开放。
二、读者一律凭本人借书证进入阅览室，在服务台自取代书板后方可选书。读者的书、包等个人物品不能带入阅览室。
三、读者在选书时，一律要用代书板，先将代书板放在所取图书的相应位置，然后取书（每次只能抽取一本书），不可乱放；阅后将书放回原位（代书板所在的位置），以免乱架。
四、读者未办理借书手续，不得擅自将书刊带出阅览室。
五、不得圈划、污损、撕页书刊。
六、不准大声喧哗，禁止吸烟、吃零食，并将手机等通讯工具调至静音，禁止在阅览区域内接打电话。
七、爱护室内财产，严禁在桌面上乱写乱刻。
八、读者应遵守阅览室规章制度，有违反上述规则者，阅览室管理人员有权进行提醒及批评教育，并按规定作相应处理。
九、本规定由图书馆负责解释。
十、本规定自公布之日起施行。
